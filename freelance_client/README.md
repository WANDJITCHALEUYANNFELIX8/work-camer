# Application d'Analyse de Données et Machine Learning (TP INF232)

Cette application est conçue pour le **Thème B (Plateforme de mise en relation freelance/client)** du TP INF232. Elle génère des données de manière déterministe et interactive, effectue les analyses statistiques univariées et bivariées, et entraîne des modèles de classification.

## Mode d'emploi (Exécution)

1. **Installer les dépendances :**
   ```bash
   pip install -r requirements.txt
   ```

2. **Générer le jeu de données :**
   ```bash
   python3 generator.py
   ```

3. **Lancer le serveur d'application :**
   ```bash
   python3 main.py
   ```

4. **Accéder à l'application :**
   Ouvrez votre navigateur web et rendez-vous sur : [http://localhost:5000](http://localhost:5000)
